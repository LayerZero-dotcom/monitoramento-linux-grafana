# Monitoramento com Grafana + Prometheus + Node Exporter

##  Objetivo
Criar um ambiente de monitoramento em Linux para coleta e visualização de métricas do sistema.

##  Tecnologias utilizadas
- Linux (Ubuntu Server)
- Prometheus
- Node Exporter
- Grafana

##  Funcionalidades
- Monitoramento de CPU
- Monitoramento de memória
- Monitoramento de disco
- Visualização em dashboard

##  Aprendizados
- Instalação e configuração de ferramentas de monitoramento
- Coleta de métricas com Prometheus
- Visualização de dados com Grafana
